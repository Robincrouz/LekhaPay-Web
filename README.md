# LekhaPay — AI Money Ledger

LekhaPay is an AI-powered financial ledger and expense tracker built for simplicity and efficiency. It helps users manage transactions, set budgets, and achieve financial goals with a smart, localized experience.

## Features

- **AI-Powered Quick Add**: Enter transactions using natural language (Bangla/English).
- **Financial Insights**: Get AI-driven advice on your spending habits.
- **Budgeting & Goals**: Set limits and track your progress towards financial freedom.
- **Invoice Management**: Professional invoice generation and tracking.
- **Progressive Web App (PWA)**: Install it on your mobile device for offline-ready access.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js (Express), Firebase (Firestore, Auth)
- **AI**: Google Gemini API
- **Charts**: Recharts

## Getting Started

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Environment Setup**: Create a `.env` file with your Gemini API Key and Firebase configuration.
4.  **Run Dev Server**:
    ```bash
    npm run dev
    ```
